# Lnt596. Minimum Subtree    
[Original Page](https://www.lintcode.com/problem/minimum-subtree/description)  

Given a binary tree, find the subtree with minimum sum. Return the root of the subtree.    

**Example 1:**  
<pre>    
Input:
     1
   /   \
 -5     2
 / \   /  \
0   2 -4  -5 

Output:1
</pre>
  
**Example 2:**  
<pre>    
Input: 1
Output: 1
</pre>
