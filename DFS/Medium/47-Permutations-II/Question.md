# 47. Permutations II  
[Original Page](https://leetcode.com/problems/permutations-ii/)

Given a collection of numbers that might **contain duplicates**, return all possible unique permutations.  

**Example:**
<pre>
Input: [1,1,2]
Output:
[
  [1,1,2],
  [1,2,1],
  [2,1,1]
]
</pre>
