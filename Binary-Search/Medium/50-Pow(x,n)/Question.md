# 50. Pow(x, n)

[Original Page](https://leetcode.com/problems/powx-n/description/)


Implement [pow(x, n)](http://www.cplusplus.com/reference/valarray/pow/).
 
**Example 1:** 
```
Input: 2.00000, 10
Output: 1024.00000
```

**Example 2:** 
```
Input: 2.10000, 3
Output: 9.26100
```
